Processus d'Utilisation
Identification :
L'utilisateur s'identifie en tant que passager ou receveur.
Code QR:
Un code QR sécurisé est généré pour chaque paiement.
Scan et Débit:
Le receveur scanne le code QR et la transaction est effectuée.
Confirmation :
La transaction est automatiquement effectuée. Les deux parties reçoivent une notification confirmant le paiement et contenant les détails de la transaction.
Modèle Économique:
- Frais de Service
KaayPas applique des frais de service de 0,7 % de la somme déposée.
- Revenus Durables :
Cette source de revenus garantit la pérennité du service tout en restant abordable pour les utilisateurs.
- Compétitivité:
